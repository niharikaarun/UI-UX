UI/UX Design

User interface is all about the looks and aesthetic of the web application whereas UX is the internal experience which the user gets after he interacts with the web application. Images can be used to depict a content in the application instead of text. We should always keep in mind that important elements can make or break the design by either reinforcing the content or derailing it and confusing the user.
